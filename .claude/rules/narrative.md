# 叙事规则 (Narrative Rules)

- 所有新增世界观内容必须与现有世界观交叉对照，检查矛盾
- 每个世界观条目必须指定正典级别：已确立 (Established) / 暂定 (Provisional) / 审查中 (Under Review)
- 角色对话必须匹配为该角色定义的语音档案
- 世界规则（什么可能/不可能）必须明确记录并保持一致
- 谜题必须有文档化的"真实答案"，即使玩家永远不会知道
- 派系动机、关系和权力结构必须在内部合乎逻辑
- 所有叙事文本必须做好本地化准备：不包含无法翻译的习语，变量使用命名占位符
- 由于对话框限制，每行对话不得超过 120 个字符

## 知识空白警告

LLM 的训练数据可能只覆盖 Godot 到 ~4.3 版本。4.4、4.5 和 4.6 引入了模型并不了解的重大变更。在建议 Godot API 调用之前，务必交叉查阅本目录。

## 截止日期之后的版本时间线

| 版本 | 发布 | 风险等级 | 关键主题 |
|---------|---------|------------|-----------|
| 4.4 | ~2025 年中 | MEDIUM | Jolt 物理选项、FileAccess 返回类型、shader 纹理类型变更 |
| 4.5 | ~2025 年末 | HIGH | 无障碍（AccessKit）、可变参数、@abstract、shader baker、SMAA |
| 4.6 | 2026 年 1 月 | HIGH | Jolt 默认、glow 重做、Windows 上 D3D12 默认、IK 恢复 |

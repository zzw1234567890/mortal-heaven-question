> **快速参考** — 层级：`经济 (Economy)` · 优先级：`Vertical Slice` · 关键依赖：`资源系统`、`卡牌系统`

## 概述

炼丹炼器系统是《仙途问道》的经济转化层，定义玩家如何将**灵材**（材料资源）加工为**丹药卡牌**和**法宝卡牌**（可使用的卡牌资产）的完整规则链——包括配方数据模型、炼制过程逻辑、品质决定机制，以及产出卡牌如何进入玩家收藏。

对玩家来说，炼丹炼器系统意味着**「创造感」**——你不再是单纯靠战斗和事件获取卡牌，而是主动选择配方、投入灵材、亲手炼制出你需要的丹药和法宝。每一次开炉都是一次小决策：是把低级灵材炼成基础丹药应急，还是攒到高级灵材再做金色法宝？这种「用双手打造卡牌」的体验，是游戏核心经济循环中区别于「战利品掉落」的另一条卡牌获取路径——偶然得来的材料，靠本事炼成有用的东西。

本系统与「法宝铭刻系统」紧密关联：炼器产出的法宝卡牌可以通过铭刻附加随机副属性，两个系统共同构成完整的法宝养成管线。

## 玩家幻想

炼丹炼器系统的核心情感是**「我亲手打造了它」的创造自豪感**——不同于战利品掉落的「运气好拿到了」，炼丹炼器的每一张卡牌都是你主动决策、投入材料、承担风险换来的成果。

| 情感 | 触发场景 |
|------|----------|
| **创造的自豪** | 灵材投入丹炉，火焰升腾——出炉一张金色丹药，「这是我做的，不是捡来的」 |
| **精打细算的权衡** | 手上有高级灵材，是做金色法宝，还是留着等材料凑齐做暗金？「再攒一攒吧……」 |
| **赌一把的刺激** | 用中等灵材尝试炼制高品质丹药——「成了就赚了，失败了也不亏太多」 |
| **量产型满足感** | 批量炼制一批基础丹药给队伍补给——「够用了，稳了」 |
| **功不唐捐的回报** | 从炼气期一路攒灵材，到金丹期终于炼出核心金卡——「这些材料没白攒」 |

**核心区别：** 战斗掉落让你感觉「被奖励了」，炼丹炼器让你感觉「我应得的」——前者是被动的运气，后者是主动的投资。这完美对齐支柱2「苟道成长」——修仙世界没有白给的资源，每一分收获都对应着你前期的积累和决策。

对百艺炼丹流玩家来说，这套系统的情感更进一层：「前期靠炼基础丹药慢慢积累属性→中期炼高级丹→后期炼顶级暗金丹」，整局游戏的成长线通过丹炉串联起来，形成一种「以丹入道」的独特修仙路径。

## 详细设计

### 核心规则

#### 1. 炼丹 (Alchemy)

**概念：** 消耗灵材炼制丹药卡牌。丹药卡牌为一次性消耗品，在战斗中使用或背包中直接使用。

##### 1a. 炼丹配方

| 配方名称 | 灵材消耗 | 产出卡牌稀有度 | 卡牌类型 |
|---------|:--------:|:--------------:|:--------:|
| 回春丹 | 低级灵材×2 | 蓝色 | 丹药卡 |
| 玉灵丹 | 中级灵材×2 + 低级灵材×1 | 紫色 | 丹药卡 |
| 天罗丹 | 高级灵材×2 + 中级灵材×1 | 金色 | 丹药卡 |
| 九转金丹 | 顶级灵材×2 + 高级灵材×1 | 暗金 | 丹药卡 |

##### 1b. 炼丹品质机制

炼丹存在「品质浮动」机制——基础产出为配方对应的稀有度，但有一定概率产出更高或更低品质。

| 因素 | 影响 | 说明 |
|------|:----:|------|
| 基础品质 | 配方固定 | 如回春丹基础为蓝色 |
| 炼丹等级提升 | 每级 +5% 高品质概率 | 炼丹等级随境界提升 |
| 万象真人本命绑定 | +15% 高品质概率 | 流派天赋联动 |
| 材料品质溢出 | 使用更高级灵材替代 → +10%/级 | 如用中级灵材替代低级灵材槽位（每槽独立计算） |

**品质概率公式：**
```
quality_roll(recipe_base_rarity, alchemy_level, bonuses):
  high_chance = 0.10 + alchemy_level × 0.05 + bonuses
  high_chance = min(high_chance, 0.8)  # 封顶80%
  
  low_chance = 0.1
  if recipe_base_rarity <= 1:  # 白色已是基础，不降品
    low_chance = 0
  
  roll = random(0, 1)
  if roll < low_chance:
    return max(recipe_base_rarity - 1, 1)  # 降品（最低稀有度=1）
  elif roll < low_chance + high_chance:
    return min(recipe_base_rarity + 1, 5)  # 升品（最高稀有度=5，暗金不可再升）
  else:
    return recipe_base_rarity      # 标准品质
```

**品质掷骰后的玩家选择（炼制主动性）：**

炼制不是纯随机——品质掷骰后，玩家有一次**主动决策**机会：

```
quality_roll 结果展示 → 玩家选择：
  [A] 接受当前品质 → 产出卡牌，炼制完成
  [B] 赌一把重掷 → high_chance +15%, low_chance +15%，重新掷骰，必须接受新结果
```

- 重掷风险：降品概率从 10% 升至 25%，但升品概率也相应提高
- 重掷不可再次取消——第二次掷骰结果必须接受
- 此设计将炼制从"拉杆老虎机"转变为"策略风险管理"——玩家看到结果后做出有意义的选择

**品质对效果的影响：**
- 降品：基础效果 × 0.8
- 标准：基础效果 × 1.0
- 升品：基础效果 × 1.3

> **关于 `bonus_pct` 参数**：`pill_effect` 公式中的 `bonus_pct`（炼丹等级≥2时+0.1）是独立于 quality_mod 的额外加成——它作用于品质修正后的结果，代表"炼丹精通"带来的效果增强。升品的 1.3 倍率中不包含此 +10%。两者是乘法叠加：`floor(base × quality_mod × (1 + bonus_pct))`。

##### 1c. 丹药效果定义

| 丹药 | 稀有度 | 效果 | 使用时机 |
|:----:|:------:|------|:--------:|
| 回春丹 | 蓝色 | 回复目标角色 4HP | 战斗中或背包 |
| 玉灵丹 | 紫色 | 回复目标角色 8HP + 驱散1个负面效果 | 战斗中或背包 |
| 天罗丹 | 金色 | 回复全体角色 6HP + 驱散全部负面效果 | 战斗中或背包 |
| 九转金丹 | 暗金 | 目标角色永久 +1 最大HP + 回复满血 | 背包（永久性） |

> 注：属性丹由修为溢出系统自动发放（`cultivation-system.md`），不走炼丹系统。

#### 2. 炼器 (Artifact Crafting)

**概念：** 消耗灵材炼制法宝卡牌。法宝卡牌为永久绑定卡，需绑定到角色后生效。

##### 2a. 炼器配方

| 配方名称 | 灵材消耗 | 产出卡牌稀有度 | 卡牌类型 |
|---------|:--------:|:--------------:|:--------:|
| 基础法器 | 低级灵材×3 | 蓝色 | 法宝卡 |
| 中品法器 | 中级灵材×3 | 紫色 | 法宝卡 |
| 上品法器 | 高级灵材×3 | 金色 | 法宝卡 |
| 通天灵宝 | 顶级灵材×3 + 高级灵材×1 | 暗金 | 法宝卡 |

##### 2b. 炼器品质机制

与炼丹相同的基础品质浮动规则（共用 `quality_roll`），但品质影响法宝白值：

- 降品：法宝基础属性 × 0.8
- 标准：法宝基础属性 × 1.0
- 升品：法宝基础属性 × 1.3

##### 2c. 法宝属性基准

```
forge_artifact_stat(rarity, quality_mod):
  base_atk = [1, 3, 4, 6, 10][rarity - 1]   # 白→暗金（白色保底 ATK=1）
  base_def = [1, 2, 3, 5, 8][rarity - 1]    # 白→暗金（白色保底 DEF=1）
  return {
    atk: max(1, floor(base_atk × quality_mod)),   # 最低 ATK=1
    def: max(0, floor(base_def × quality_mod))    # DEF 允许为 0
  }
```

| 稀有度 | 基准攻击 | 基准防御 | 降品 (×0.8) | 标准 (×1.0) | 升品 (×1.3) |
|:-----:|:-------:|:-------:|:-----------:|:-----------:|:-----------:|
| 白色 | 1 | 1 | —（不降品） | 1/1 | 2/1 |
| 蓝色 | 3 | 2 | 2/1 | 3/2 | 3/2 |
| 紫色 | 4 | 3 | 3/2 | 4/3 | 5/3 |
| 金色 | 6 | 5 | 4/4 | 6/5 | 7/6 |
| 暗金 | 10 | 8 | 8/6 | 10/8 | 13/10 |

> **设计说明**：白色法宝基准 ATK=1/DEF=1（而非 0/0），确保降品法宝仍有最低战术价值——降品损失了潜力，但不会得到一张完全无用的卡牌。蓝色基准 ATK 从 2 提升至 3，使降品 ×0.8=2（有感知的削弱），且升品通过稀有度提升获得实质差异。
>
> 注：法宝的具体技能/特效由 `card-system.md` 的 `equip_effect` 字段管理，本系统只负责攻防白值生成。

##### 2d. 铭刻接口

铭刻消耗由「法宝铭刻系统」GDD 权威定义：

```
inscribe_cost(artifact_inscription_count) → (item, qty):
  N = artifact_inscription_count + 1
  return (中级灵材, N)  # 递增：第1次=1, 第2次=2, 第3次=3...
```

#### 3. 炼丹/炼器等级

- 初始等级 = 0
- 每突破一个大境界 → 等级 +1
- 最大等级 = 4（化神期）

**配方解锁条件：**

| 配方 | 所需等级 | 对应境界 |
|:----:|:--------:|:--------:|
| 回春丹 / 基础法器 | 0 | 炼气（初始可用） |
| 玉灵丹 / 中品法器 | 1 | 筑基 |
| 天罗丹 / 上品法器 | 2 | 金丹 |
| 九转金丹 / 通天灵宝 | 3 | 元婴 |

**等级 4（化神期）特殊能力——「丹道大成」：**

达到炼丹/炼器等级 4（化神期）后解锁被动能力：**每局首次炼丹/炼器必定升品**（跳过 quality_roll，直接产出升品结果）。此能力在每局游戏中生效一次——第二次及以后的炼制按正常品质规则。

> **设计理由**：化神期是游戏终点境界。在此之前，玩家已通过 3 个境界解锁全部配方、积累了大量灵材和炼丹经验。化神期的「丹道大成」是对玩家长期修炼的肯定——不是数值上的 +5% 概率，而是一次确定性的"你可以完美炼制一次"。这与玩家幻想「功不唐捐的回报」完美契合。

#### 4. 炼制位置

- 探索地图中：**炼器台/丹炉节点** → 访问后打开炼制界面
- 营寨中：**丹房**功能 → 与探索节点共用规则

炼丹炼器系统无独立状态机。其生命周期跟随玩家的操作流程：

```

                              灵材不足 → 返回配方选择（灰显提示材料不足）

```

## 公式

### 0. 品质掷骰→品质倍率映射

```
quality_roll_result → quality_mod:
  降品（rarity 降低 1）  → quality_mod = 0.8
  标准（rarity 不变）    → quality_mod = 1.0
  升品（rarity 提升 1）  → quality_mod = 1.3
```

### 1. 品质概率公式

```
quality_roll(recipe_base, alchemy_level, bonuses) → int:
  high_chance = min(0.10 + alchemy_level × 0.05 + bonuses, 0.8)
  low_chance = (recipe_base > 1) ? 0.1 : 0
  roll = random(0, 1)
  if roll < low_chance:
    return max(recipe_base - 1, 1)   # 降品（下限=1）
  if roll < low_chance + high_chance:
    return min(recipe_base + 1, 5)   # 升品（上限=5，暗金不可再升）
  return recipe_base                  # 标准品质
```

| 变量 | 类型 | 范围 | 描述 |
|------|------|:----:|------|
| recipe_base | int | [1, 5] | 配方基础品质（白=1→暗金=5） |
| alchemy_level | int | [0, 4] | 当前炼丹/炼器等级 |
| bonuses | float | [0, 0.45] | 外部加成（万象真人+0.15 + 材料溢出每槽+0.10） |
| high_chance | float | [0.10, 0.8] | 升品概率（封顶80%） |
| low_chance | float | [0, 0.1] | 降品概率 |

**示例：** 金丹期（等级=2）无加成炼制回春丹（base=2）→ high_chance=20%，20%升品紫，10%降品白，70%标准蓝

### 1b. 品质重掷公式

```
reroll_quality(recipe_base, alchemy_level, bonuses) → int:
  # 玩家选择重掷后调用
  high_chance = min(0.10 + alchemy_level × 0.05 + bonuses + 0.15, 0.8)
  low_chance = (recipe_base > 1) ? 0.25 : 0   # 降品概率升至 25%
  roll = random(0, 1)
  if roll < low_chance:
    return max(recipe_base - 1, 1)
  if roll < low_chance + high_chance:
    return min(recipe_base + 1, 5)
  return recipe_base
```

| 变量 | 类型 | 范围 | 描述 |
|------|------|:----:|------|
| high_chance | float | [0.25, 0.8] | 重掷升品概率（基础+15%加成） |
| low_chance | float | [0, 0.25] | 重掷降品概率固定 25% |

### 2. 丹药效果缩放

```
pill_effect(base_value, quality_mod, bonus_pct) → int:
  # bonus_pct 是独立的"炼丹精通"加成，与 quality_mod 乘法叠加
  return max(1, floor(base_value × quality_mod × (1 + bonus_pct)))
```

| 变量 | 类型 | 范围 | 描述 |
|------|------|:----:|------|
| base_value | int | 配方固定 | 丹药基础效果值 |
| quality_mod | float | {0.8, 1.0, 1.3} | 品质倍率 |
| bonus_pct | float | [0, 0.1] | 炼丹精通加成（炼丹等级≥2时+0.1） |

**九转金丹特殊处理：** 永久+1最大HP部分的 `base_value`=1，通过 `pill_effect` 缩放的公式同上。降品时 `max(1, floor(1×0.8)) = 1`（保底+1最大HP——降品不消除金丹的标志性永久效果）。恢复满血部分不经过 `pill_effect` 公式——它独立结算（品质不影响恢复量）。

### 3. 法宝属性生成

```
forge_artifact_stat(rarity, quality_mod) → {atk, def}:
  base_atk = [1, 3, 4, 6, 10][rarity - 1]   # 白→暗金
  base_def = [1, 2, 3, 5, 8][rarity - 1]    # 白→暗金
  return {
    atk: max(1, floor(base_atk × quality_mod)),   # 最低 ATK=1
    def: max(0, floor(base_def × quality_mod))    # DEF 允许为 0
  }
```

| 变量 | 类型 | 范围 | 描述 |
|------|------|:----:|------|
| rarity | int | [1, 5] | 产出稀有度索引（白=1→暗金=5） |
| quality_mod | float | {0.8, 1.0, 1.3} | 品质倍率 |

**完整产出矩阵：**
| 稀有度 | 基准 ATK/DEF | 降品 (×0.8) | 标准 (×1.0) | 升品 (×1.3) |
|:-----:|:-----------:|:-----------:|:-----------:|:-----------:|
| 白（1） | 1/1 | — | 1/1 | 2/1 |
| 蓝（2） | 3/2 | 2/1 | 3/2 | 3/2 |
| 紫（3） | 4/3 | 3/2 | 4/3 | 5/3 |
| 金（4） | 6/5 | 4/4 | 6/5 | 7/6 |
| 暗金（5） | 10/8 | 8/6 | 10/8 | 13/10 |

### 4. 拆解炼制物价值

```
dismantle_crafted_value(card) → int:
  # 炼制物拆解折价 50%——防止炼制→拆解套利
  standard_dismantle = [10, 30, 100, 400, 2000][card.rarity - 1]
  if card.is_crafted:
    return floor(standard_dismantle × 0.5)
  return standard_dismantle
```

| 变量 | 类型 | 范围 | 描述 |
|------|------|:----:|------|
| card.is_crafted | bool | — | 是否由炼制系统产出（标记在卡牌实例数据中） |
| standard_dismantle | int | [10, 2000] | 标准拆解价值 |
| 返回值 | int | [5, 1000] | 炼制物拆解灵石（折价50%） |

**示例：** 拆解炼制的暗金法宝 → floor(2000×0.5)=1000灵石（vs 灵材成本680灵石，+47%而非之前的+194%）

### 5. 九转金丹递减收益

```
jindan_permanent_hp_bonus(craft_count_this_run) → int:
  # 每局限递减：第1颗+1HP, 第2颗需累计3颗, 第3颗需累计6颗...
  # 实际增益始终为 +1HP，但所需炼制次数递增
  required = craft_count_this_run × (craft_count_this_run + 1) / 2
  # 第1次: 需要累计1颗 → +1HP
  # 第2次: 需要累计3颗（1+2） → 再+1HP
  # 第3次: 需要累计6颗（1+2+3） → 再+1HP
  return 1  # 每次突破门槛固定+1HP
```

> **设计意图**：不做硬上限（允许深度百艺流玩家追求极致），但成本指数增长使"无限刷HP"不现实——第5次+1HP需要累计15颗九转金丹（15×480=7200灵石）。这是软上限而非硬禁止。

## 边界情况

- **灵材不足时选择配方**：配方选项显示为灰色，标注缺少的灵材种类和数量，不可点击
- **炼制中途退出**：玩家选择配方但未确认炼制前可以自由退出，不消耗任何灵材
- **品质掷骰后重掷选择**：quality_roll 结果展示后，玩家可选择接受或重掷。重掷后必须接受新结果，不可再次取消。重掷时降品概率升至 25%
- **升品/降品/标准的动画反馈**：每种结果有不同的视觉反馈（升品有额外光效和闪光）；降品损失了潜力但不产出无用品（白色法宝保底 ATK=1/DEF=1）
- **炼丹等级0时尝试高等级配方**：配方在列表中可见但标注「突破至筑基后可炼」，不可选
- **背包中已有同名丹药但丹药卡有stack_limit(3)**：由卡牌系统的 stack_limit 约束管理，炼制产出超过 stack_limit 时收藏池正常增加（收藏池允许同名多张）
- **多份同种丹药同时炼制**：当前设计不支持批量炼制——每次开炉固定炼制1份。（化神期「丹道大成」能力不提供批量功能，仅保底一次升品）
- **已满级后继续提升等级**：炼丹等级达到4（化神）后不再增长；化神期解锁「丹道大成」——每局首次炼制必定升品
- **万象真人阵亡后加成消失**：绑定万象真人的角色阵亡后，品质加成立即失效。炼制时如果万象真人不处于上场/存活状态，bonuses 不包含+15%
- **降品至白色**：降品结果始终是 recipe_base - 1（最低=1）。蓝色配方降品得白色（蓝→白）；白色法宝保底 ATK=1/DEF=1，非 0/0
- **暗金配方降品**：九转金丹配方（暗金）降品→金色，永久+1HP 部分保底 1（降品不消除金丹标志性效果）；通天灵宝降品→金色
- **暗金配方升品**：quality_roll 已将上限钳制为 5（暗金），暗金配方升品结果仍为暗金——即"升品"对暗金配方无效（相当于标准品质）。玩家应优先在暗金配方上选择接受而非重掷
- **炼制界面不消耗行动力**：在探索地图丹炉节点炼器不消耗行动力（但访问丹炉本身的移动消耗了行动力）
- **同一地图多次使用炼器台**：不限制次数——灵材足够即可炼制
- **重掷后品质掷骰结果与初始相同**：理论可能（概率低）——玩家承担了 +15% 额外降品风险却未获得升品。这是重掷的风险成本，系统不做"保底与初始不同"
- **炼制物拆解折价**：所有经由炼制系统产出的卡牌标记 `is_crafted=true`，拆解时灵石收益为同稀有度标准拆解价值的 50%。拆解炼制物不返还灵材——仅返还折价灵石
- **九转金丹递减收益**：第1次炼制+1HP。第2次+1HP需累计3颗。第3次+1HP需累计6颗。累计计数跨战斗/跨地图保存，随存档持久化

## 验收标准

### 核心炼制流程

- **AC-1**：**GIVEN** 玩家拥有低级灵材×2、收藏有空位，**WHEN** 选择炼制回春丹并确认，**THEN** 低级灵材-2，获得稀有度为蓝色的「回春丹」丹药卡进入收藏（效果：回复目标角色 4HP）
- **AC-2**：**GIVEN** 玩家灵材不足（低级灵材<2），**WHEN** 选择回春丹配方，**THEN** 确认按钮灰显且不可点击，配方旁标注「缺少：低级灵材×2」
- **AC-3**：**GIVEN** 玩家炼丹等级=0（炼气期），**WHEN** 查看配方列表，**THEN** 玉灵丹、天罗丹、九转金丹配方灰显，各配方标注「需要炼丹等级 N（境界名称）方可解锁」，其中 N=1:"筑基"、2:"金丹"、3:"元婴"
- **AC-4**：**GIVEN** 玩家已突破至筑基，**WHEN** 境界提升后首次打开炼制界面，**THEN** 显示横幅通知「玉灵丹配方已解锁」，持续 3 秒或直到被关闭

### 品质浮动

- **AC-5**：**GIVEN** 玩家炼制回春丹且品质掷骰结果为升品，**WHEN** 结算完成，**THEN** 获得稀有度为紫色的回春丹，效果为 floor(4×1.3) = 5HP 回复
- **AC-6**：**GIVEN** 玩家炼制回春丹且品质掷骰结果为降品，**WHEN** 结算完成，**THEN** 获得稀有度为白色的回春丹，效果为 max(1, floor(4×0.8)) = 3HP 回复
- **AC-7**：**GIVEN** 玩家炼制金色上品法器且品质掷骰为标准，**WHEN** 查看产出法宝属性，**THEN** ATK=6、DEF=5
- **AC-8**：**GIVEN** 玩家炼制金色上品法器且品质掷骰为升品，**WHEN** 查看产出法宝属性，**THEN** ATK=7、DEF=6
- **AC-8a**：**GIVEN** 玩家炼制蓝色基础法器且品质掷骰为降品（→白色），**WHEN** 查看产出法宝属性，**THEN** ATK=max(1, floor(1×0.8))=1、DEF=max(0, floor(1×0.8))=0
- **AC-8b**：**GIVEN** 玩家炼制蓝色基础法器且品质掷骰为升品（→紫色），**WHEN** 查看产出法宝属性，**THEN** ATK=floor(4×1.3)=5、DEF=floor(3×1.3)=3
- **AC-8c**：**GIVEN** 玩家炼制通天灵宝（暗金）且品质掷骰为升品，**WHEN** 结算，**THEN** 产出稀有度仍为暗金（上限 5），法宝属性 ATK=10、DEF=8（升品对暗金无效）

### 品质重掷

- **AC-8d**：**GIVEN** 玩家炼制回春丹后 quality_roll 结果为标准品质，**WHEN** 选择重掷，**THEN** high_chance 增加 15%、low_chance 升至 25%，必须接受新掷骰结果
- **AC-8e**：**GIVEN** 玩家炼制回春丹后 quality_roll 结果为降品，选择重掷，**WHEN** 重掷结果仍为降品，**THEN** 玩家获得降品丹药（接受结果，无第三次重掷机会）

### 等级与解锁

- **AC-9**：**GIVEN** 玩家突破至金丹期（境界 L=3），**WHEN** 检查炼丹等级，**THEN** 等级=2（=境界层级-1）
- **AC-9a**：**GIVEN** 玩家达到化神期（境界 L=5），**WHEN** 检查炼丹等级，**THEN** 等级=4，且「丹道大成」能力解锁——本局首次炼制必定升品
- **AC-9b**：**GIVEN** 化神期玩家已在本局使用过「丹道大成」必升品，**WHEN** 进行第二次炼制，**THEN** 按正常 quality_roll 规则处理（不保底）

### 暗金与降品

- **AC-10**：**GIVEN** 玩家炼制九转金丹且品质掷骰为降品，**WHEN** 结算，**THEN** 获得稀有度为金色的「九转金丹」，永久+1HP 部分保底为 1（max(1, floor(1×0.8))=1），回复满血效果不受品质影响
- **AC-10a**：**GIVEN** 玩家炼制通天灵宝且品质掷骰为降品，**WHEN** 查看产出法宝属性，**THEN** 稀有度=金色，ATK=max(1, floor(6×0.8))=4、DEF=max(0, floor(5×0.8))=4

### 外部加成与炼制物拆解

- **AC-11**：**GIVEN** 万象真人绑定角色已阵亡，玩家炼丹等级=0，**WHEN** 品质掷骰的 high_chance 被计算，**THEN** high_chance = 10%（0.10 + 0×0.05 + 0，不含 +15% 万象真人加成）
- **AC-11a**：**GIVEN** 玩家拆解一张炼制产出的回春丹（is_crafted=true），**WHEN** 结算，**THEN** 获得灵石 = floor(30×0.5) = 15（标准拆解价值 50%）
- **AC-11b**：**GIVEN** 玩家拆解一张炼制产出的通天灵宝（暗金，is_crafted=true），**WHEN** 结算，**THEN** 获得灵石 = floor(2000×0.5) = 1000

### 集成测试

- **AC-12**：**GIVEN** 玩家在地图丹炉节点拥有 5 行动力，**WHEN** 完成一次回春丹炼制并返回配方选择，**THEN** 玩家仍拥有 5 行动力（炼制本身不消耗行动力）
- **AC-13**：**GIVEN** 玩家收藏中已有一张回春丹，炼制并获得第二张回春丹，**WHEN** 打开卡组编辑界面，**THEN** 收藏池中出现两张独立回春丹卡牌（收藏池允许同名多张）
- **AC-14**：**GIVEN** 玩家已炼制过 1 次九转金丹（jindan_count=1），**WHEN** 检查九转金丹永久 HP 增益要求，**THEN** 第 2 次 +1HP 需要累计 3 颗（1+2），当前仍需额外 2 颗
- **AC-15**：**GIVEN** 玩家品质掷骰后选择重掷，**WHEN** 重掷结果展示，**THEN** 接受按钮可用但无重掷按钮（不可再次重掷）
- **AC-16**：**GIVEN** 化神期玩家首次打开炼制界面，**WHEN** 检查丹道大成状态，**THEN** 显示「丹道大成·可用」（本局首次炼制必定升品）

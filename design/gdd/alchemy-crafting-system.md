> **状态 (Status)**：设计中
> **作者 (Author)**：Claude Code + 用户

> **快速参考** — 层级：`经济 (Economy)` · 优先级：`Vertical Slice` · 关键依赖：`资源系统`、`卡牌系统`

## 概述

炼丹炼器系统是《凡人修仙传：掌天问道》的经济转化层，定义玩家如何将**灵材**（材料资源）加工为**丹药卡牌**和**法宝卡牌**（可使用的卡牌资产）的完整规则链——包括配方数据模型、炼制过程逻辑、品质决定机制，以及产出卡牌如何进入玩家收藏。

对玩家来说，炼丹炼器系统意味着**「创造感」**——你不再是单纯靠战斗和事件获取卡牌，而是主动选择配方、投入灵材、亲手炼制出你需要的丹药和法宝。每一次开炉都是一次小决策：是把低级灵材炼成基础丹药应急，还是攒到高级灵材再做金色法宝？这种「用双手打造卡牌」的体验，是游戏核心经济循环中区别于「战利品掉落」的另一条卡牌获取路径——偶然得来的材料，靠本事炼成有用的东西。

本系统与「法宝铭刻系统」紧密关联：炼器产出的法宝卡牌可以通过铭刻附加随机副属性，两个系统共同构成完整的法宝养成管线。

## 玩家幻想

炼丹炼器系统的核心情感是**「我亲手打造了它」的创造自豪感**——不同于战利品掉落的「运气好拿到了」，炼丹炼器的每一张卡牌都是你主动决策、投入材料、承担风险换来的成果。

| 情感 | 触发场景 |
|------|----------|
| **创造的自豪** | 灵材投入丹炉，火焰升腾——出炉一张金色丹药，「这是我做的，不是捡来的」 |
| **精打细算的权衡** | 手上有高级灵材，是做金色法宝，还是留着等材料凑齐做暗金？「再攒一攒吧……」 |
| **赌一把的刺激** | 用中等灵材尝试炼制高品质丹药——「成了就赚了，失败了也不亏太多」 |
| **量产型满足感** | 批量炼制一批基础丹药给队伍补给——「够用了，稳了」 |
| **功不唐捐的回报** | 从炼气期一路攒灵材，到金丹期终于炼出核心金卡——「这些材料没白攒」 |

**核心区别：** 战斗掉落让你感觉「被奖励了」，炼丹炼器让你感觉「我应得的」——前者是被动的运气，后者是主动的投资。这完美对齐支柱2「苟道成长」——修仙世界没有白给的资源，每一分收获都对应着你前期的积累和决策。

对百艺炼丹流玩家来说，这套系统的情感更进一层：「前期靠炼基础丹药慢慢积累属性→中期炼高级丹→后期炼顶级暗金丹」，整局游戏的成长线通过丹炉串联起来，形成一种「以丹入道」的独特修仙路径。

## 详细设计

### 核心规则

#### 1. 炼丹 (Alchemy)

**概念：** 消耗灵材炼制丹药卡牌。丹药卡牌为一次性消耗品，在战斗中使用或背包中直接使用。

##### 1a. 炼丹配方

| 配方名称 | 灵材消耗 | 产出卡牌稀有度 | 卡牌类型 |
|---------|:--------:|:--------------:|:--------:|
| 回春丹 | 低级灵材×2 | 蓝色 | 丹药卡 |
| 玉灵丹 | 中级灵材×2 + 低级灵材×1 | 紫色 | 丹药卡 |
| 天罗丹 | 高级灵材×2 + 中级灵材×1 | 金色 | 丹药卡 |
| 九转金丹 | 顶级灵材×2 + 高级灵材×1 | 暗金 | 丹药卡 |

##### 1b. 炼丹品质机制

炼丹存在「品质浮动」机制——基础产出为配方对应的稀有度，但有一定概率产出更高或更低品质：

| 因素 | 影响 | 说明 |
|------|:----:|------|
| 基础品质 | 配方固定 | 如聚气散基础为蓝色 |
| 炼丹等级提升 | 每级 +5% 高品质概率 | 炼丹等级随境界提升 |
| 大衍神君本命绑定 | +15% 高品质概率 | 流派天赋联动 |
| 材料品质溢出 | 使用更高级灵材替代 → +10%/级 | 如用中级灵材替代低级部位 |

**品质概率公式：**
```
quality_roll(recipe_base_rarity, alchemy_level, bonuses):
  high_chance = 0.05 + alchemy_level × 0.05 + bonuses
  high_chance = min(high_chance, 0.8)  # 封顶80%
  
  low_chance = 0.1
  if recipe_base_rarity <= 1:  # 白色已是基础，不降品
    low_chance = 0
  
  roll = random(0, 1)
  if roll < low_chance:
    return recipe_base_rarity - 1  # 降品
  elif roll < low_chance + high_chance:
    return recipe_base_rarity + 1  # 升品
  else:
    return recipe_base_rarity      # 标准品质
```

**品质对效果的影响：**
- 降品：基础效果 × 0.8
- 标准：基础效果 × 1.0
- 升品：基础效果 × 1.3（含高品质额外+10%）

##### 1c. 丹药效果定义

| 丹药 | 稀有度 | 效果 | 使用时机 |
|:----:|:------:|------|:--------:|
| 回春丹 | 蓝色 | 回复目标角色 4HP | 战斗中或背包 |
| 玉灵丹 | 紫色 | 回复目标角色 8HP + 驱散1个负面效果 | 战斗中或背包 |
| 天罗丹 | 金色 | 回复全体角色 6HP + 驱散全部负面效果 | 战斗中或背包 |
| 九转金丹 | 暗金 | 目标角色永久 +1 最大HP + 回复满血 | 背包（永久性） |

> 注：属性丹由修为溢出系统自动发放（`cultivation-system.md`），不走炼丹系统。

---

#### 2. 炼器 (Artifact Crafting)

**概念：** 消耗灵材炼制法宝卡牌。法宝卡牌为永久绑定卡，需绑定到角色后生效。

##### 2a. 炼器配方

| 配方名称 | 灵材消耗 | 产出卡牌稀有度 | 卡牌类型 |
|---------|:--------:|:--------------:|:--------:|
| 基础法器 | 低级灵材×3 | 蓝色 | 法宝卡 |
| 中品法器 | 中级灵材×3 | 紫色 | 法宝卡 |
| 上品法器 | 高级灵材×3 | 金色 | 法宝卡 |
| 通天灵宝 | 顶级灵材×3 + 高级灵材×1 | 暗金 | 法宝卡 |

##### 2b. 炼器品质机制

与炼丹相同的基础品质浮动规则（共用 `quality_roll`），但品质影响法宝白值：

- 降品：法宝基础属性 × 0.8
- 标准：法宝基础属性 × 1.0
- 升品：法宝基础属性 × 1.3

##### 2c. 法宝属性基准

```
forge_artifact_stat(rarity, quality_mod):
  base_atk = [0, 2, 4, 6, 10][rarity - 1]   # 白→暗金
  base_def = [0, 1, 3, 5, 8][rarity - 1]    # 白→暗金
  return {
    atk: floor(base_atk × quality_mod),
    def: floor(base_def × quality_mod)
  }
```

| 稀有度 | 基准攻击 | 基准防御 | 降品 | 升品 |
|:-----:|:-------:|:-------:|:----:|:----:|
| 蓝色 | 2 | 1 | 1/1 | 2/1 |
| 紫色 | 4 | 3 | 3/2 | 5/3 |
| 金色 | 6 | 5 | 4/4 | 7/6 |
| 暗金 | 10 | 8 | 8/6 | 13/10 |

> 注：法宝的具体技能/特效由 `card-system.md` 的 `equip_effect` 字段管理，本系统只负责攻防白值生成。

##### 2d. 铭刻接口

铭刻消耗在本系统定义，具体规则由「法宝铭刻系统」GDD 展开：

```
inscribe_cost() → int:
  return 1  # 中级灵材×1，每次铭刻消耗不变
```

#### 3. 炼丹/炼器等级

- 初始等级 = 0
- 每突破一个大境界 → 等级 +1
- 最大等级 = 4（化神期）

**配方解锁条件：**

| 配方 | 所需等级 | 对应境界 |
|:----:|:--------:|:--------:|
| 回春丹 / 基础法器 | 0 | 炼气（初始可用） |
| 玉灵丹 / 中品法器 | 1 | 筑基 |
| 天罗丹 / 上品法器 | 2 | 金丹 |
| 九转金丹 / 通天灵宝 | 3 | 元婴 |

#### 4. 炼制位置

- 探索地图中：**炼器台/丹炉节点** → 访问后打开炼制界面
- 营寨中：**丹房**功能 → 与探索节点共用规则

### 状态与转换

炼丹炼器系统无独立状态机。其生命周期跟随玩家的操作流程：

```
[进入炼制界面] → [选择配方] → [检查灵材库存]
                                     ↓
                              灵材不足 → 返回配方选择（灰显提示材料不足）
                                     ↓
                              灵材充足 → [确认炼制]
                                           ↓
                           扣减灵材 → 执行 quality_roll()
                                           ↓
                             产出卡牌 → [展示结果动画]
                                           ↓
                             卡牌存入收藏 → [返回配方选择或退出]
```

## 公式

### 1. 品质概率公式

```
quality_roll(recipe_base, alchemy_level, bonuses) → int:
  high_chance = min(0.05 + alchemy_level × 0.05 + bonuses, 0.8)
  low_chance = (recipe_base > 1) ? 0.1 : 0
  roll = random(0, 1)
  if roll < low_chance:     return recipe_base - 1
  if roll < low_chance + high_chance: return recipe_base + 1
  return recipe_base
```

| 变量 | 类型 | 范围 | 描述 |
|------|------|:----:|------|
| recipe_base | int | [1, 5] | 配方基础品质（白=1→暗金=5） |
| alchemy_level | int | [0, 4] | 当前炼丹/炼器等级 |
| bonuses | float | [0, 0.3] | 外部加成（大衍神君+0.15等） |
| high_chance | float | [0.05, 0.8] | 升品概率（封顶80%） |
| low_chance | float | [0, 0.1] | 降品概率 |

**示例：** 金丹期（等级=2）无加成炼制回春丹（base=2）→ high_chance=15%，15%升品紫，10%降品白，75%标准蓝

### 2. 丹药效果缩放

```
pill_effect(base_value, quality_mod, bonus_pct) → int:
  return floor(base_value × quality_mod × (1 + bonus_pct))
```

| 变量 | 类型 | 范围 | 描述 |
|------|------|:----:|------|
| base_value | int | 配方固定 | 丹药基础效果值 |
| quality_mod | float | [0.8, 1.3] | 品质倍率（降品0.8/标准1.0/升品1.3） |
| bonus_pct | float | [0, 0.1] | 高品质额外加成（炼丹等级≥2时+0.1） |

### 3. 法宝属性生成

```
forge_artifact_stat(rarity, quality_mod) → {atk, def}:
  base_atk = [0, 2, 4, 6, 10][rarity - 1]
  base_def = [0, 1, 3, 5, 8][rarity - 1]
  return {atk: floor(base_atk × quality_mod), def: floor(base_def × quality_mod)}
```

| 变量 | 类型 | 范围 | 描述 |
|------|------|:----:|------|
| rarity | int | [2, 5] | 产出稀有度索引（蓝=2→暗金=5） |
| quality_mod | float | [0.8, 1.3] | 品质倍率（降品0.8/标准1.0/升品1.3） |

**示例：** 炼制金色上品法器且升品（quality_mod=1.3）→ ATK=floor(6×1.3)=7, DEF=floor(5×1.3)=6

## 边界情况

- **灵材不足时选择配方**：配方选项显示为灰色，标注缺少的灵材种类和数量，不可点击
- **炼制中途退出**：玩家选择配方但未确认炼制前可以自由退出，不消耗任何灵材
- **升品/降品/标准的动画反馈**：每种结果有不同的视觉反馈（升品有额外光效和闪光），降品不惩罚玩家（只是不出高品质）
- **炼丹等级0时尝试高等级配方**：配方在列表中可见但标注「突破至筑基后可炼」，不可选
- **背包中已有同名丹药但丹药卡有stack_limit(3)**：由卡牌系统的 stack_limit 约束管理，炼制产出超过 stack_limit 时收藏池正常增加（收藏池允许同名多张）
- **多份同种丹药同时炼制**：当前设计不支持批量炼制——每次开炉固定炼制1份
- **已满级后继续提升等级**：炼丹等级达到4（化神）后不再增长，所有配方已解锁
- **大衍神君阵亡后加成消失**：绑定大衍神君的角色阵亡后，品质加成立即失效。炼制时如果大衍神君不处于上场/存活状态，bonuses 不包含+15%
- **降品至白色**：降品结果始终是 recipe_base - 1。蓝色配方降品得白色（蓝→白）
- **暗金配方降品**：九转金丹配方（暗金）降品→金色；通天灵宝降品→金色
- **炼制界面不消耗行动力**：在探索地图丹炉节点炼器不消耗行动力（但访问丹炉本身的移动消耗了行动力）
- **同一地图多次使用炼器台**：不限制次数——灵材足够即可炼制

## 调优参数

| 参数 | 默认值 | 安全范围 | 过低影响 | 过高影响 |
|------|:-----:|:--------:|---------|---------|
| 升品基础概率 | 5% | 5%~15% | 品质浮动无感 | 高品质通胀 |
| 升品每级加成 | +5% | +3%~+10% | 等级提升无回报感 | 高等级必定升品 |
| 升品概率封顶 | 80% | 60%~90% | 高等级升级无意义 | 必定升品失去随机性 |
| 降品概率 | 10% | 5%~20% | 无风险感 | 玩家不敢炼高级配方 |
| 升品效果倍率 | 1.3 | 1.2~1.5 | 升品回报不够 | 升品=碾压 |
| 降品效果倍率 | 0.8 | 0.7~0.9 | 降品惩罚过重 | 降品无感 |
| 大衍神君品质加成 | +15% | +10%~+25% | 天赋感觉不明显 | 百艺流派必定升品 |
| 丹药基础效果（聚气散） | 4HP | 3~6HP | 回复不够难救急 | 替代战斗奶妈 |
| 丹药基础效果（真元丹） | 8HP+驱散1 | 6~10HP | 性价比不如聚气散 | 聚气散无意义 |
| 法宝蓝色基准ATK | 2 | 1~3 | 炼器不如不掉落 | 法宝太强本体不重要 |
| 法宝暗金基准ATK | 10 | 8~12 | 暗金法宝不够强 | 暗金=无敌 |
| 暗金法宝基准DEF | 8 | 6~10 | 不够肉 | 打不动

## 验收标准

- **GIVEN** 玩家有低级灵材×2，**WHEN** 选择炼制聚气散并确认，**THEN** 低级灵材-2，获得蓝色丹药「聚气散」入收藏
- **GIVEN** 玩家灵材不足，**WHEN** 选择配方，**THEN** 确认按钮灰显并标注缺少的灵材
- **GIVEN** 玩家炼丹等级=0，**WHEN** 查看配方列表，**THEN** 真元丹及以上配方显示为灰显并标注解锁条件
- **GIVEN** 玩家在炼气期突破至筑基后，**WHEN** 打开炼制界面，**THEN** 显示「真元丹配方已解锁」
- **GIVEN** 玩家炼制聚气散且触发升品，**WHEN** 结算，**THEN** 产出紫色丹药（真元丹效果×1.3）
- **GIVEN** 玩家炼制聚气散且触发降品，**WHEN** 结算，**THEN** 产出白色丹药（聚气散效果×0.8）
- **GIVEN** 玩家炼制金色上品法器且标准品质，**WHEN** 查看产出法宝属性，**THEN** ATK=6、DEF=5
- **GIVEN** 玩家炼制金色上品法器且升品，**WHEN** 查看产出法宝属性，**THEN** ATK=7、DEF=6
- **GIVEN** 玩家突破至金丹期，**WHEN** 检查炼丹等级，**THEN** 等级=2
- **GIVEN** 玩家炼制九转金丹且降品，**WHEN** 结算，**THEN** 产出金色丹药（天罗丹效果×1.3而非暗金）
- **GIVEN** 大衍神君绑定角色已阵亡，**WHEN** 玩家炼制，**THEN** 品质概率不包含+15%加成
- **GIVEN** 玩家在地图丹炉节点，**WHEN** 使用炼制功能，**THEN** 不消耗行动力
- **GIVEN** 玩家炼制完成获得卡牌，**WHEN** 打开卡组编辑界面，**THEN** 新卡牌出现在收藏池中

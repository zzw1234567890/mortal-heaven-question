## 电梯演讲 (Elevator Pitch)

> 这是一款Roguelike卡牌对战游戏，你作为一名修仙者在苍玄大陆上自由构筑卡组、探索随机地图、经历原创剧情，通过「步步攀升、苟道成长」的方式从炼气期一路渡劫飞升。

## 核心幻想 (Core Fantasy)

你是一个在残酷修仙世界中从零起步的凡人。没有逆天天赋，没有背景靠山——你唯有靠自己的智慧和一点点机缘，步步为营地成长。每一场战斗、每一次探索、每一个决定，都让你离「飞升」更近一步，也可能让你一朝陨落。这就是修仙的魅力：不是天选之子的爽快，而是凡人靠努力和运气逆天改命的成就感。

---

## 独特卖点 (Unique Hook)

这是一款Roguelike卡牌游戏，**以及**它用「功法/法宝绑定角色」的卡牌机制而非传统的「打出即生效」，让每张卡牌在构筑之外多了一层战术绑定决策；**以及**它将修仙的「渡劫突破」作为核心成长循环——攒修为→渡天劫→开新地图，每次突破带来质变。

## 玩家体验分析 — MDA 框架 (Player Experience Analysis)

### 目标审美（玩家感受到什么）(Target Aesthetics)

| 审美 | 优先级 | 我们如何实现 |
| ---- | ---- | ---- |
| **挑战 (Challenge)**（障碍课程、精通） | 1 | 渡劫突破需要策略性备战，境界压制机制要求玩家合理规划路线 |
| **发现 (Discovery)**（探索、秘密） | 2 | 随机地图事件、洞府奇遇、夺舍选择、隐藏三星洞 |
| **幻想 (Fantasy)**（假装、角色扮演） | 3 | 原创修仙世界、代入角色、体验从凡人到化神的成长 |
| **叙事 (Narrative)**（戏剧、故事弧线） | 4 | 5大章节原创剧情，结局分支提供重玩动力 |
| **表达 (Expression)**（自我表达、创造力） | 5 | 5种流派、30-40张自由组牌、本命绑定加成支持多样构筑 |
| **感觉 (Sensation)**（感官愉悦） | 6 | 卡牌特效、境界突破动画、法术相克视觉效果 |
| **沉浸 (Submission)**（放松、舒适区） | N/A | 游戏强调策略挑战而非放松体验 |
| **伙伴 (Fellowship)**（社交连接） | N/A | 单人游戏，无社交功能 |

### 关键动态（涌现的玩家行为）(Key Dynamics)

- 玩家会围绕流派核心卡牌构建卡组，试验不同角色+功法+法宝的组合协同
- 玩家会策略性地规划探索路线：是稳健发育还是冒险越阶挑战
- 玩家会在「快速推进剧情」和「慢慢积累修为」之间权衡
- 玩家会因为不同的初始身份和流派偏好而体验截然不同的构筑风格

### 核心机制（我们构建的系统）(Core Mechanics)

1. **回合制卡牌对战** — 费用管理、角色上场、功法/法宝绑定、阵法规制
2. **地图随机探索** — 行动力消耗、6种随机事件、资源获取与风险决策
3. **修为养成与渡劫突破** — 修为积累→渡劫挑战→境界提升→解锁新内容
4. **自由卡组构建** — 30-40张卡组、5种卡牌类型、本命加成、多流派支持
5. **炼丹炼器制造** — 灵材→丹药/法宝制作、法宝铭刻、副属性系统

### 玩家类型吸引力（Bartle 分类）

- [x] **成就者 (Achievers)** — 收集全部222张卡牌、完成所有渡劫、解锁全结局

## 核心循环 (Core Loop)

### 瞬间到瞬间（30 秒）(Moment-to-Moment)
在地图上移动 → 触发事件 → 判断是否战斗 → 回合制出牌战斗 → 获得资源/卡牌

### 短期（5-15 分钟）(Short-Term)
完成一个探索区域 → 触发奇遇或战斗 → 调整卡组/炼丹炼器 → 回复行动力

### 单次会话级（30-120 分钟）(Session-Level)
从当前地图开始 → 探索+战斗+养成循环 → 攒够修为触发渡劫 → 突破大境界或战败损失修为

### 长期成长 (Long-Term Progression)
炼气→筑基→金丹→元婴→化神 五大境界逐级突破 → 解锁新地图和新卡牌掉落池 → 面对更强的Boss → 通关/开始新局（保留部分轮回天赋）

### 留存钩子 (Retention Hooks)
- **好奇心**：随机事件的不确定性、隐藏奇遇、未解锁的卡牌
- **投入感**：修为积累、卡组构筑的心血、流派成型
- **精通**：越阶渡劫挑战、更高难度、5种流派的深度

## MVP 定义 (MVP Definition)

**核心假设**：玩家在「卡牌构筑×修为成长×渡劫突破」的循环中找到重复游玩的乐趣。

**MVP 必需**：
1. 核心战斗系统（回合制卡牌对战、费用、角色上场、绑定、阵法）
2. 一个完整境界（炼气期）+ 渡劫突破到筑基
3. 基础卡牌池（约60张覆盖基本流派）
4. 随机探索事件（3种基本事件类型）
5. 基础卡组构建功能

**明确不在 MVP 中**：
- 全部222张卡牌（MVP约60张）
- 炼丹炼器系统（简化成商店购买）
- 全结局分支（仅基础剧情线）

## 核心幻想 (Core Fantasy)

你是一个在残酷修仙世界中从零起步的凡人。没有逆天天赋，没有背景靠山——你唯有靠自己的智慧和一点点机缘，步步为营地成长。每一次突破，都是逆天改命的一步。

| ---- | ---- | ---- |

## 支柱 (The Pillars)

### 支柱 1：「自由组牌，策略为王」

**一句话定义**：卡组构筑是玩家表达策略的核心方式——每张卡牌的效果必须明确可预测，流派之间有清晰的协同路径，但不存在单一的最优解。

**服务的目标审美**：表达 (Expression)、挑战 (Challenge)

**设计测试**：当玩家研究构筑时，是应该能预期卡牌间的协同效应（法阵+功法+角色的联动），还是随机试错？答案：可预期的协同。

#### 这对每个部门意味着什么

| 部门 | 此支柱要求... | 示例 |
| ---- | ---- | ---- |
| **游戏设计** | 每张卡牌有明确的流派归属，流派间有区别但不过度强化单一方向 | 正道=防御续航，魔道=快攻杀人，互有优劣 |
| **美术** | 卡牌视觉效果能帮助玩家快速识别卡牌类型和稀有度 | 功法/法宝/角色卡的视觉区分+稀有度颜色编码 |
| **音频** | 不同流派的卡牌有对应的风格化音效 | 魔道卡暗沉低音，正道卡清亮高亢 |
| **叙事** | 本命加成机制贴合世界观的角色关系逻辑 | 林渊绑定枯木逢春诀有额外加成，而非随机搭配 |
| **工程** | 卡牌效果解析引擎必须是数据驱动的，方便扩展和平衡调整 | 222张卡牌的效果由数据配置而非硬编码 |

#### 服务于本支柱
- 30-40张自由组牌空间，支持多套卡组预设
- 「本命绑定加成」鼓励但强制特定搭配——玩家可以选择提升羁绊或自由发挥
- 5种流派各有清晰的卡牌核心和阵法支持

#### 违反本支柱
- 某张单卡强度远超同稀有度的选择（降低构筑多样性）
- 存在对所有流派都「必带」的通用核心卡
- 某流派被版本答案完全压制，无法通关

### 支柱 2：「苟道成长，步步为营」

**一句话定义**：玩家的成长曲线是渐进式的——每一次渡劫突破都应带来质变，但玩家需要为此做足准备；不鼓励无脑冲锋，而鼓励策略性规划路线。

**服务的目标审美**：挑战 (Challenge)、幻想 (Fantasy)

**设计测试**：玩家在面对境界压制时，是应该感觉「我被不公平对待了」还是「我该先去突破再回来打」？答案：后者——境界压制鼓励思考路线规划。

#### 这对每个部门意味着什么

| 部门 | 此支柱要求... | 示例 |
| ---- | ---- | ---- |
| **游戏设计** | 境界压制有明确数值但不过分；渡劫失败不致命（损失10%修为）；越阶渡劫有额外奖励 | 伤害减20%/50%但要限，而非100%不可打 |
| **美术** | 境界突破有显著的视觉仪式感 | 天劫特效、突破动画、修为条积累的视觉反馈 |
| **音频** | 境界突破和渡劫有专门的音效/音乐 | 渡劫时紧张音乐vs突破后的成就感音乐 |
| **叙事** | 每个境界对应一段剧情成长，角色对话反映实力变化 | 回到旧地图时NPC反应你的境界提升 |
| **工程** | 修为/境界是核心数据模型，所有系统通过境界获取上下文 | 卡牌掉落池、事件难度、敌人AI均受境界影响 |

#### 服务于本支柱
- 境界压制规则：高1级减20%，高2级减50%——让玩家有压力但不绝望
- 渡劫失败只损失10%修为，可以重试
- 越阶渡劫奖励额外金卡，鼓励敢冒险的玩家
- 修为溢出自动转化为属性丹（不浪费）
- 行动力自然回复+突破回满

#### 违反本支柱
- 玩家可以在不突破的情况下通关整个游戏
- 渡劫失败导致游戏重置（一着不慎满盘皆输）
- 境界提升只是数值增加，没有任何新内容解锁

### 支柱 3：「机缘巧合，意外之喜」

**一句话定义**：探索阶段充满了随机事件和隐藏机缘——玩家不是简单地图上走格子，而是每次移动都面临有趣的选择和意想不到的发现。

**服务的目标审美**：发现 (Discovery)、叙事 (Narrative)

**设计测试**：玩家在探索时，是应该只走最短路程打Boss通关，还是每次探索都有期待发现的惊喜？答案：随机事件让每局游戏的探索体验独一无二。

#### 这对每个部门意味着什么

| 部门 | 此支柱要求... | 示例 |
| ---- | ---- | ---- |
| **游戏设计** | 随机事件有明确的3选1结构；事件之间有关联线索；隐藏事件需要特定条件触发 | 三星洞隐藏奇遇、夺舍事件的高低风险决策 |
| **美术** | 每个事件类型有独特的场景插图和氛围 | 灵脉采掘的山水场景 vs 坊市交易的热闹集市 |
| **音频** | 不同类型事件有不同的环境音 | 洞府的幽静风铃 vs 战斗的杀伐之音 |
| **叙事** | 随机事件中包含原创世界观的梗和角色彩蛋 | 墨渊夺舍事件、万象真人残魂事件 |
| **工程** | 事件系统是数据驱动的，支持条件触发、连锁事件和结果分支 | 事件表可配置，支持概率权重和条件检查 |

#### 服务于本支柱
- 6种随机事件类型：灵脉采掘、坊市交易、洞府奇遇、杀人夺宝、炼丹/炼器台、三星洞
- 每张地图的三星洞隐藏奇遇（每图仅触发1次）
- 夺舍事件的2选1决策：高风险高回报
- 杀人夺宝的PvE遭遇：打赢得全部，打输损失一半

#### 违反本支柱
- 事件选择毫无意义（不管选什么结果都一样）
- 随机事件太少，探索阶段变得可预测且乏味
- 存在「选必赢」或「选必输」的事件选项

### 支柱 4：「修仙问道，唯我独行」

**一句话定义**：游戏构建了一个完整的原创修仙世界——苍玄大陆上正魔对峙、上古归墟文明埋藏秘密、每个角色的命运由玩家的选择决定。没有固定剧本——你的每一次旅途本身就是一部修仙传记。

**服务的目标审美**：幻想 (Fantasy)、叙事 (Narrative)

**设计测试**：玩家看到「林渊+青锋逐影剑+青云剑诀」的组合时，是否会自动产生「这就是剑修之路」的认同感？答案：本命加成机制和角色设计就是要实现这种情感回报。

#### 这对每个部门意味着什么

| 部门 | 此支柱要求... | 示例 |
| ---- | ---- | ---- |
| **游戏设计** | 角色、功法、法宝、事件均基于原创世界观；本命加成强化角色组合 | 青云剑诀+青锋逐影剑绑定林渊效果翻倍 |
| **美术** | 角色立绘和场景贴合原创世界观描述 | 林渊外貌、青云剑宗场景、碎星群岛风格 |
| **音频** | 主要角色有专属配音或音乐主题 | 林渊、月清霜、汐音等核心角色的主题音效 |
| **叙事** | 5章剧情跟随原创剧情线：青云入世→碎星流亡→苍玄之争→归墟探索→问道终局 | 每章末尾有原创结局分支选择 |
| **工程** | 本命绑定加成表可配置，支撑后续内容更新 | 数据配置的角色+功法+法宝协同关系 |

#### 服务于本支柱
- 角色设计贴合原创世界观：墨渊的夺舍特性、林渊的苟道天赋
- 混沌宝瓶作为核心机缘贯穿游戏
- 章节顺序：青云入世→碎星流亡→苍玄之争→归墟探索→问道终局
- 角色卡牌的设计和阵营标签基于原创势力划分

#### 违反本支柱
- 角色能力和世界观设定完全无关
- 出现世界观中不存在的高阶角色或跨界联动
- 剧情走向脱离世界观框架到无法识别的程度

## 反支柱（本游戏**不是**什么）(Anti-Pillars)

- **不是开放世界RPG**：游戏采用章节式地图+格子探索，没有自由漫游开放世界。这个约束保护「机缘巧合」支柱——有边界的探索才能做到事件密度和设计质量。
- **不是社交/多人游戏**：纯单机体验，无合作、对战或社区功能。这保护「自由组牌」支柱——不需要考虑PVP平衡性，可以专注于策略多样的PVE体验。
- **不是动作游戏**：回合制卡牌对战，没有动作要素（无实时反应、无操作精度要求）。这保护「策略为王」支柱——胜负取决于构筑和决策而非手速。
- **不是故事驱动的叙事游戏**：剧情是核心体验的支撑而非核心玩法。玩家不会观看大量过场动画——故事通过卡牌、事件和探索对话来传递。

## 支柱冲突解决 (Pillar Conflict Resolution)

| 优先级 | 支柱 | 理由 |
| ---- | ---- | ---- |
| 1 | **自由组牌，策略为王** | 没有好的策略深度，其他支柱没有落脚点 |
| 2 | **苟道成长，步步为营** | 核心Roguelike成长循环是玩家持续游玩的动力 |
| 3 | **机缘巧合，意外之喜** | 随机事件是Roguelike重玩性的关键差异点 |
| 4 | **修仙问道，唯我独行** | 原创世界观增强幻想和叙事，但不约束机制设计 |

**解决流程**：
1. 识别哪些支柱处于紧张状态
2. 查阅上述优先级排名
3. 如果低优先级支柱可以在不损害高优先级支柱的情况下部分满足，则这样做
4. 如果不能，高优先级支柱胜出
5. 在相关设计文档中记录决策和理由
6. 如果冲突是根本性的，上报给创意总监

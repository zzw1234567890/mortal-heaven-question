## 概述

凡人的成长哲学是「物尽其用」——一件法宝到手只是开始，通过铭刻附加副属性，让同一件法宝在玩家手中产生差异化的战术价值，是「我亲手打造的法宝」而非「系统掉落的标准品」。

## 玩家幻想

「我的法宝和你的不一样」——铭刻系统服务的核心情感是**养成专属感**。同一种法宝，在你手中通过铭刻长出了不同的副属性组合，变成了属于你构筑的一部分。这是一种「养孩子」式的投入感——你不只是在收集法宝，你是在培养它们。

| 情感 | 触发场景 |
|------|----------|
| **专属感** | 「我这个上品法器是+2攻击+吸血的——你的呢？」 |
| **精炼的满足** | 三条铭刻全都洗到了有用的属性上——「完美了」 |
| **不强求的豁达** | 铭刻出废属性——「算了，这条先留着，下次再洗」 |
| **成长的期待** | 刚炼出一件好法宝，还有三个铭刻空位可以慢慢培养 |

## 详细设计

### 核心规则

#### 1. 铭刻基本规则

- 每件法宝初始拥有 **0 条副属性**，最多可铭刻 **3 条**
- 每次铭刻消耗 **中级灵材×1**，立即附加 **1 条随机副属性**
- 铭刻次数不受限制——已满 3 条后可继续铭刻，**随机替换已有 1 条副属性**（覆盖重随）
- 铭刻操作不可逆——选定即生效，无法撤销

#### 2. 副属性池

| 副属性 | 效果 | 初始权重 | 品质梯级 |
|--------|------|:--------:|:--------:|
| 攻击+1 | 角色攻击力+1 | 22 | T1 |
| 防御+1 | 角色防御力+1 | 18 | T1 |
| 暴击+3% | 暴击概率+3%（加法叠加） | 15 | T2 |
| 暴击伤害+5% | 暴击时额外+5%伤害 | 12 | T2 |
| 生命+2 | 角色最大生命+2 | 10 | T3 |
| 吸血+2 | 造成伤害时回复等量HP | 8 | T3 |
| 「虚弱」 | 攻击时使目标本回合伤害-1（可叠加层数） | 6 | T3 |
| 费用-1 | 该法宝卡费用-1（最低为1） | 4 | T4（稀有） |
| 回合开始回HP+1 | 装备角色每回合开始回复1HP | 3 | T4（稀有） |
| 「破甲」 | 攻击时使目标本回合防御-2（可叠加层数） | 3 | T4（稀有） |
| 「灵力萃取」 | 装备角色每回合开始获得+1临时灵力（仅本回合可用） | 2 | T4（稀有） |

#### 3. 品质梯级与权重规则

副属性按稀有度分为 T1~T4 四个梯级：
- **T1**（攻击/防御）：最常见的基础数值
- **T2**（暴击/暴伤）：中频概率属性
- **T3**（生命/吸血/虚弱）：较稀有，偏战术效果
- **T4**（费用-1/回合回血/破甲/灵力萃取）：极其稀有，改变战斗节奏

**权重获取规则：**
```
roll_substat(existing_substats, realm_level):
  weights = {
    攻击+1:22, 防御+1:18,
    暴击+3%:15, 暴伤+5%:12,
    生命+2:10, 吸血+2:8, 虚弱:6,
    费用-1:4, 回HP+1:3, 破甲:3, 灵力萃取:2
  }
  
  # 已有相同副属性时权重减半（防止极端堆积）
  for each stat in existing_substats:
    if stat in weights:
      weights[stat] = floor(weights[stat] × 0.5)
  
  # 境界提升增加稀有属性权重
  bonus = floor(realm_level × 2)  # 炼气+2, 筑基+4, 金丹+6, 元婴+8
  weights["费用-1"] += bonus
  weights["回HP+1"] += bonus
  weights["破甲"] += bonus
  weights["灵力萃取"] += bonus
  
  return weighted_random(weights)
```

#### 4. 铭刻对已满法宝的处理

当法宝已有 3 条副属性时再次铭刻：
1. 玩家选择消耗中级灵材×1 → 进入替换选择界面
2. 玩家**手动选择 1 条已有副属性**作为被替换目标
3. 按权重 roll 新副属性（排除被替换的那条，但可能 roll 到其他已有属性）
4. 替换生效，不可撤销

> **设计理由：** 让玩家选择替换哪条属性而非随机，玩家不会因误随机洗掉极品属性而受挫——这尊重玩家对法宝的养成投入。

#### 5. 法宝升级与铭刻的关系

- 法宝卡牌升级（消耗灵石+同名卡）不改变已有铭刻
- 铭刻属性在法宝升级后保留——副属性与等级独立成长
- 卡牌拆解时，铭刻副属性不影响拆解价值（拆解只看稀有度和等级）

### 状态与转换

```
法宝状态: [无铭刻] → 铭刻1次 → [1条副属性] → 铭刻1次 → [2条副属性] → 铭刻1次 → [3条副属性]
                                                                           ↓
                                                                     [3条满] → 铭刻→玩家选择替换1条
```

## 公式

### 1. 副属性权重投掷

```
roll_substat(existing_substats, realm_level) → substat:
  base_weights = {
    "atk+1":22, "def+1":18,
    "crit+3":15, "crit_dmg+5":12,
    "hp+2":10, "lifesteal+2":8, "weakness":6,
    "cost-1":4, "regen+1":3, "armor_break":3, "mana_extract":2
  }
  
  for stat in existing_substats:
    if stat in base_weights:
      base_weights[stat] = floor(base_weights[stat] × 0.5)
  
  bonus = floor(realm_level × 2)
  base_weights["cost-1"] += bonus
  base_weights["regen+1"] += bonus
  base_weights["armor_break"] += bonus
  base_weights["mana_extract"] += bonus
  
  return weighted_random(base_weights)
```

| 变量 | 类型 | 范围 | 描述 |
|------|------|:----:|------|
| existing_substats | string[] | [0, 3] | 当前法宝已有副属性列表 |
| realm_level | int | [1, 5] | 当前境界层级 |
| bonus | int | [2, 10] | 稀有属性权重加成 |

**示例：** 金丹期（等级=3）玩家铭刻金色法宝，已有「攻击+1」→ 攻击权重从22降至11

### 2. 替换目标选择（3条满时）

```
select_replacement_target(existing_substats, player_choice) → int:
  # 玩家从3条中选择1条作为替换目标
  # 返回被选中副属性的索引
  player_choice ∈ [0, 1, 2]
  return player_choice
```

### 3. 铭刻费用

```
inscribe_cost() → (item, qty):
  return (中级灵材, 1)
```

## 边界情况

- **首次铭刻**：法宝首次铭刻无特殊规则——直接 roll 新属性，加入 substats[]
- **3条满后铭刻**：弹出替换选择界面，玩家从现有3条副属性中选1条作为被替换目标；玩家选定的那条将被新副属性覆盖
- **玩家反悔**：在替换选择界面玩家可以取消操作，不消耗灵材
- **所有副属性完全相同（3条攻击+1）**：允许——概率极低但理论上可行。这是玩家可以自豪的「极品法宝」
- **费用-1叠加**：费用-1不叠加——即使有2条费用-1也只生效1条，法宝最低1费
- **回合回HP叠加**：可叠加——2条「回合开始回HP+1」=每回合回2HP
- **吸血叠加**：可叠加——2条「吸血+2」=造成伤害时回复4HP
- **虚弱叠加**：可叠加——攻击时叠加层数。2条虚弱=每次攻击使目标本回合伤害-2
- **破甲叠加**：可叠加——攻击时叠加层数。2条破甲=每次攻击使目标本回合防御-4
- **灵力萃取叠加**：可叠加——2条灵力萃取=每回合开始获得+2临时灵力（仅本回合可用）
- **虚弱与破甲持续回合**：debuff效果仅持续到敌人本回合结束（与「本回合」绑定），不跨回合累积
- **灵力萃取的临时灵力**：获得的灵力仅限本回合使用，回合结束时清空。不影响下回合基础灵力
- **铭刻时灵材不足**：提示「需要中级灵材×1」，不可操作
- **铭刻后立即生效**：铭刻完成后不重新抽牌——当前战斗中已抽到的法宝不受新铭刻影响，下一场战斗生效
- **铭刻不消耗行动力**：在营寨中或探索界面铭刻时不消耗任何行动力

## 验收标准

- **GIVEN** 玩家有中级灵材×1和无铭刻的法宝，**WHEN** 点击铭刻，**THEN** 扣除灵材×1，法宝获得1条随机副属性
- **GIVEN** 玩家铭刻时已有3条副属性，**WHEN** 点击铭刻，**THEN** 显示替换选择界面列出3条副属性，玩家选择1条后确认，灵材消耗，替换生效
- **GIVEN** 玩家在替换选择界面，**WHEN** 点击取消，**THEN** 不消耗灵材，返回法宝详情
- **GIVEN** 玩家法宝已有「攻击+1」，**WHEN** roll副属性时，**THEN** 攻击+1的权重从22降至11
- **GIVEN** 玩家境界=金丹(L=3)，**WHEN** roll副属性时，**THEN** 费用-1/回血/破甲/灵力萃取各+6权重
- **GIVEN** 玩家法宝有「吸血+2」，**WHEN** 装备角色造成伤害，**THEN** 回复等量HP
- **GIVEN** 玩家法宝有「虚弱」，**WHEN** 攻击敌人，**THEN** 敌人本回合伤害-1
- **GIVEN** 玩家法宝有「破甲」，**WHEN** 攻击敌人，**THEN** 敌人本回合防御-2
- **GIVEN** 玩家法宝有「灵力萃取」，**WHEN** 回合开始，**THEN** 获得+1临时灵力（仅本回合可用）
- **GIVEN** 玩家法宝有2条「灵力萃取」，**WHEN** 回合开始，**THEN** 获得+2临时灵力
- **GIVEN** 玩家灵材不足，**WHEN** 尝试铭刻，**THEN** 铭刻按钮灰显并提示灵材不足
- **GIVEN** 玩家铭刻后，**WHEN** 查看法宝详情，**THEN** 新副属性已写入并显示
